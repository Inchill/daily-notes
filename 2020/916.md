## 使用typescript时出现Property 'timer' does not exist on type 'Vue'

vscode 提示vue对象下没有定义的属性和方法(其实是已经定义过了)。

这样写不会提示错误：
```js
export default {
  data() {
    return {
      timer: null,
    };
  },
  methods: {
    run() {
      (this as any).timer = window.setInterval(() => {
        console.log('1');
      }, 1000);
    },
    stop() {
      clearInterval((this as any).timer);
    },
  },
};
```
